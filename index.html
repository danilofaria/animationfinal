COMS W4167: Computer Animation - Fall 2014, Final Project

Columbia University, The Fu Foundation School of Engineering and Applied Science

Student: Danilo Faria (df2553)

I made use of the Three.js library to render my scenes, but I coded all the physics and program logics from scratch. This project is my first experience with Three.js. The code I implemented is in the folder "/js/mycode/". The physics, integrators and scene class are in the file NOTFSim.js (90125 Simulator).

Scenes Secondary Text

Scene One

Solar System

A simple model of our Solar System. Utilizes gravitational forces that I've implemented in javascript based on our previous course experience. All planets are to scale with each other in size but not in distances (however, the sun is not).

View Scene

Scene Two

Gravity and Spring

In this scene you can see several "planets" going in random directions around a center body. The center body consists of a bunch of particles connected by springs to one central particle.

Scene Three

Springs

Buoyancy

Planet Buoyancy
